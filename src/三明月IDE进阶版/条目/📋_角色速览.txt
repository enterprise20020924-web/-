角色总览指南

任务说明：
- 检索之前创作的所有角色
- 整理成简单的列表格式
- 只包含基本识别信息
- 方便AI快速定位角色
- 用代码块包裹输出

需要检索的标签：
- <sample_basic> - 角色基础信息
- <npc_1>, <npc_2> 等 - NPC角色
- <user_setting> - {{user}}设定
- 其他包含角色信息的标签

工作流程：

第一步：检索所有角色
- 在聊天记录中找到所有角色相关标签
- 提取基本识别信息
- 包括主角、配角、NPC

第二步：整理格式
- 按角色重要性排序
- 只保留最基础的识别信息
- 姓名、性别、年龄、身份/职务/班级
- 必要时添加备注

第三步：输出
- 用代码块包裹
- 简洁的列表格式
- 方便快速查找定位

输出格式：

```
角色总览:
  - 姓名: [名字]
    性别: [性别]
    年龄: [年龄]
    身份: [学生/教师/职务等]
    班级: [如果是学生]
    职务: [如果有职务]
    备注: [必要的额外信息]

  - 姓名: [名字]
    性别: [性别]
    年龄: [年龄]
    ...

[如果有背景设定，可以添加]
背景信息:
  学校名称: [学校名]
  地点: [地点]
  时代: [时代背景]
  ...
```

整理原则：

1. 极简原则
- 只保留识别信息
- 不包含性格、外貌等详细内容
- 方便AI快速定位"谁是谁"

2. 关键信息
- 必须：姓名、性别、年龄
- 可选：身份、职务、班级、关系
- 备注：重要的识别信息

3. 清晰排序
- {{user}}放最前（如果有）
- 主要角色在前
- NPC在后
- 便于查找

示例：

假设创作了多个角色，总览应该是：

```
角色总览:
  - 姓名: '{{user}}'

    备注: 与{{user}}同班

  - 姓名: 王老师
    性别: 女
    年龄: 35
    职务: 班主任
    负责班级: 高二3班

  - 姓名: 小林
    性别: 男
    年龄: 17
    班级: 高二3班
    备注: {{user}}的同桌

学校信息:
  学校名称: 市立第三高中
  年级设置: 高一、高二、高三
  学制: 三年制
```

AI的输出方式：

这是当前的角色总览：

```
[总览内容]
```

方便你快速定位每个角色的基本信息。需要添加或修改可以告诉我。

重要提醒：
- 这是角色索引，不是详细人设
- 只包含基本识别信息
- 方便AI快速定位"这是谁"
- 用代码块包裹
- 按重要性排序

完成后的引导

当角色速览整理完成后：

1. 使用 Write 工具将角色速览写入用户世界书：
   - 路径：/Worldbooks/<用户世界书名>/角色速览
2. 使用 SetAttribute 工具为该条目设置默认世界书配置：
   - strategy: { type: "constant" }
   - position: { type: "before_character_definition", order: 4 }
   - recursion: { prevent_incoming: true, prevent_outgoing: true }
3. 使用 Read 工具读取世界书中名为"写卡知识库"的世界书中的"一般条目自查"条目（重点参考"角色卡制作审查标准"部分，额外注意角色速览特有要求：信息是否极简、有无冗余详情）
4. 按照自查标准逐项检查刚才的输出，发现问题直接修改并输出修正后的完整版本
5. 修改完成后询问用户："是否需要将修正后的内容同步更新到世界书对应条目？"
   - 如果用户同意，使用 Edit 工具更新世界书中的条目
6. 输出提醒：

哥哥，角色速览已完成并写入世界书，条目配置已自动设置！

下一步：创作开场白。对我说"写开场白"。
