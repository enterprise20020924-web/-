世界书条目标签包裹规范

核心规则：
- 每个世界书条目的内容必须用 <名称_idN> 标签包裹
- N 是递增的数字ID，按以下规则分配
- 标签用于帮助AI区分不同条目的内容来源和层级关系
- 同一角色的所有条目（基础信息、调色盘、二次解释、衣柜、NSFW等）共用同一个标签ID

==========

【ID分配顺序】

整体分配优先级（从小到大）：
1. 世界观条目（id1~idX）
2. 角色速览（紧随世界观之后）
3. 各角色条目（按创建顺序分配，同角色共用ID）
4. NPC条目（最后分配）

==========

【世界观标签】

按世界观层级分配ID：

路径A/B（单条目世界观）：
- 只有一个世界观条目时，只用 <世界观_id1>

路径C（大世界多条目）：
- <世界观_id1> — 世界总纲/总设定
- <世界观_id2> — 区域速览/背景设定
- <世界观_id3> — 各势力/场景详情（可有多个条目，都用id3）
- <世界观_id4> — 各事件类型

==========

【角色速览标签】

- <角色速览_id0> — 角色总览条目
- 角色速览使用id0，不占用正式ID序列

==========

【角色标签】

核心规则：同一角色的所有条目共用同一个标签ID

- 同一角色的基础信息、调色盘人设、三面性、二次解释、衣柜、NSFW调色盘等，虽然是不同的世界书条目，但写入时都用同一个 <角色名_idN> 标签包裹
- 不同角色用不同ID
- 条目内部不再额外嵌套标签
- ID按角色创建顺序递增分配

示例：
- 秋明月是第一个创建的角色 → 她的所有条目都用 <秋明月_id1>
- 林小雨是第二个创建的角色 → 她的所有条目都用 <林小雨_id2>

注意：如果世界观占用了id1~id4，则第一个角色从id5开始

==========

【NPC标签】

- 每个NPC独立一个标签
- 格式：<NPC名_idN>
- ID在角色之后继续递增
- 每个NPC只有一个条目，所以每个NPC一个ID

==========

【完整示例】

假设一张角色卡包含：
- 大世界设定（路径C）：总纲、区域速览、2个势力详情、1个事件
- 角色速览
- 2个主要角色：秋青子、夏橘
- 3个NPC：王老师、小林、店长

则ID分配如下：

世界观：
  <世界观_id1> — 世界总纲
  <世界观_id2> — 区域速览
  <世界观_id3> — 势力A详情
  <世界观_id3> — 势力B详情（同层级，共用id3）
  <世界观_id4> — 事件类型

角色速览：
  <角色速览_id0> — 角色总览

角色（从id5开始，因为世界观用了id1~id4）：
  <秋青子_id5> — 秋青子基础信息
  <秋青子_id5> — 秋青子调色盘人设（同角色，同ID）
  <秋青子_id5> — 秋青子二次解释（同角色，同ID）
  <秋青子_id5> — 秋青子衣柜（同角色，同ID）
  <夏橘_id6> — 夏橘基础信息
  <夏橘_id6> — 夏橘调色盘人设（同角色，同ID）

NPC（继续递增）：
  <王老师_id7> — 王老师
  <小林_id8> — 小林
  <店长_id9> — 店长

---

假设一张简单的单角色卡：
- 真实背景世界观（路径A）：1个条目
- 角色速览
- 1个主要角色：秋青子
- 1个NPC：小林

则ID分配如下：

世界观：
  <世界观_id1> — 总设定

角色速览：
  <角色速览_id0> — 角色总览

角色（从id2开始）：
  <秋青子_id2> — 秋青子基础信息
  <秋青子_id2> — 秋青子调色盘人设（同角色，同ID）
  <秋青子_id2> — 秋青子二次解释（同角色，同ID）

NPC：
  <小林_id3> — 小林

==========

【写入格式】

写入世界书时，条目内容的实际格式为：

```
<角色名_idN>
（条目正文内容）
</角色名_idN>
```

例如秋青子的基础信息条目写入世界书时：
```
<秋青子_id2>
角色档案:
  基本信息:
    姓名: 秋青子
    ...
</秋青子_id2>
```

秋青子的调色盘条目写入世界书时：
```
<秋青子_id2>
性格调色盘:人的性格就像调色盘...
...
</秋青子_id2>
```

==========

【重要提醒】

1. 标签ID一旦分配，在整个角色卡生命周期内保持不变
2. 新增条目时，根据类型找到对应的ID规则：
   - 新增同角色条目 → 使用该角色已有的ID
   - 新增新角色 → 在当前最大角色ID后+1
   - 新增NPC → 在当前最大ID后+1
3. 删除条目不影响其他条目的ID（不重新编号）
4. 同层级的世界观详情条目共用ID（如多个势力都用id3）
5. 角色速览固定使用id0，不参与递增序列
