衣柜模板

核心原则：
- 列出角色拥有的服装，不规定具体穿着
- 让AI根据场景自由搭配
- 用具体描述，不用抽象词
- 符合角色身份和经济状况

衣柜结构：

一、日常穿着

  上衣类:
    - 列出具体款式和颜色
    - T恤、衬衫、毛衣等
    - 颜色、图案、特征

  下装类:
    - 裤子、裙子
    - 款式、长度、颜色

  外套类:
    - 夹克、大衣、开衫
    - 季节、厚薄、颜色

二、特殊场合

  正式场合:
    - 有什么正式服装
    - 套装、礼服等

  运动场合:
    - 运动服、运动鞋

  居家服:
    - 睡衣、家居服
    - 宽松舒适的衣服

三、内衣袜子

  内衣:
    - 款式类型
    - 颜色偏好
    - 不写具体哪天穿哪件

  袜子:
    - 长袜、短袜
    - 颜色和款式

  其他:
    - 打底裤等

四、鞋子

  日常鞋:
    - 运动鞋、休闲鞋
    - 颜色和款式

  特殊鞋:
    - 皮鞋、高跟鞋
    - 拖鞋、凉鞋

五、配饰

  常用配饰:
    - 帽子、围巾、手套
    - 包包
    - 首饰

  发饰:
    - 发卡、发圈
    - 头绳

写作要点：

只列出拥有的，不规定穿着:
  错误方式:
    - "周一穿白色T恤和牛仔裤"
    - "正式场合穿黑色套装"
    - "在家穿粉色睡衣"

  正确方式:
    - 有几件白色T恤，有图案的和纯色的
    - 有一套黑色西装套装，正式场合可以穿
    - 有粉色、蓝色、白色的睡衣

用具体描述:
  不够具体:
    - "几件衬衫"
    - "一些裙子"
    - "运动鞋"

  足够具体:
    - 白色衬衫、格子衬衫、蓝色衬衫各一件
    - 黑色A字短裙、灰色长裙、碎花连衣裙
    - 白色运动鞋、黑色帆布鞋

符合身份和经济:
  学生角色:
    - 以校服为主
    - 休闲服装
    - 不会有太贵的衣服

  工作角色:
    - 职业装
    - 日常休闲装
    - 符合职业特点

  富裕角色:
    - 品牌服装
    - 款式更多
    - 质量更好

季节考虑:
  春秋:
    - 长袖T恤、衬衫
    - 薄外套、开衫
    - 长裤、中长裙

  夏季:
    - 短袖、吊带
    - 短裤、短裙
    - 连衣裙

  冬季:
    - 毛衣、厚外套
    - 羽绒服、大衣
    - 长裤、打底裤

示例：

这是衣柜清单的参考。只列出拥有的衣服，不规定怎么穿，AI会自己搭配：

```
<sample_wardrobe>
林小雨的衣柜:

  日常上衣:
    T恤:
      - 白色短袖T恤两件，纯色的
      - 浅蓝色T恤一件，领口有小蝴蝶结
      - 粉色长袖T恤一件，有点旧了
      - 灰色T恤一件，宽松款

    衬衫:
      - 白色长袖衬衫一件，校服款式
      - 格子衬衫一件，红白格子的

    毛衣:
      - 米色毛衣一件，圆领的
      - 浅灰色开衫一件，有口袋
      - 粉色针织衫一件，比较薄

  日常下装:
    裤子:
      - 蓝色牛仔裤两条，一条浅一条深
      - 黑色长裤一条，上学穿的
      - 灰色运动裤一条，宽松的

    裙子:
      - 校服裙子，深蓝色，到膝盖
      - 白色棉布长裙一条，到脚踝
      - 浅蓝色A字裙一条，到膝盖上

  外套:
    春秋外套:
      - 米色风衣一件，有点大
      - 牛仔外套一件，浅蓝色
      - 黑色薄夹克一件

    冬季外套:
      - 深蓝色羽绒服一件，到大腿
      - 灰色大衣一件，到膝盖

  内衣:
    内衣:
      - 白色棉质内衣几套
      - 粉色、浅蓝色内衣各一套
      - 都是简单款式，没有蕾丝
      - 罩杯不大，A杯的

    内裤:
      - 白色、粉色、浅蓝色的
      - 都是纯棉的
      - 简单款式

  袜子:
    - 白色长筒袜好几双，上学穿
    - 黑色短袜几双
    - 粉色、白色的可爱短袜
    - 有一双有兔子图案的

  鞋子:
    - 黑色皮鞋一双，上学穿的
    - 白色运动鞋一双，有点旧了
    - 粉色帆布鞋一双
    - 居家拖鞋，毛茸茸的，粉色

  睡衣:
    - 粉色睡衣套装，上面有小熊图案
    - 白色睡裙，到膝盖
    - 浅蓝色睡衣，长袖长裤的

  配饰:
    - 黑色书包，上学用的
    - 帆布小挎包，米色的
    - 黑色发圈几个
    - 白色蝴蝶结发卡一个
    - 粉色发夹几个

  其他:
    - 黑色打底裤两条
    - 围巾一条，米色的，冬天用
    - 手套一副，粉色的
</sample_wardrobe>
```

重要提醒：
- 只列出拥有什么，不规定怎么穿
- 让AI根据场景自由搭配
- 描述要具体：颜色、款式、特征
- 符合角色身份和经济状况
- 考虑季节因素
- 数量要合理，不是越多越好
- 避免八股化描写
- 保持绝对零度和白描

完成后的引导

当衣柜模板创作完成后：

1. 使用 Write 工具将衣柜写入用户世界书：
   - 路径：/Worldbooks/<用户世界书名>/角色/<角色名>/衣柜
2. 使用 SetAttribute 工具为该条目设置默认世界书配置：
   - 单角色卡：
     strategy: { type: "constant" }
     position: { type: "after_character_definition", order: 99 }
   - 多角色卡：
     strategy: { type: "selective", keys: ["角色名"] }
     position: { type: "after_character_definition", order: 99 }
   - 通用：recursion: { prevent_incoming: true, prevent_outgoing: true }
3. 使用 Read 工具读取世界书中名为"写卡知识库"的世界书中的"一般条目自查"条目
4. 按照自查标准逐项检查：确保描述具体、符合角色身份，发现问题直接修改并输出修正后的完整版本
5. 修改完成后询问用户："是否需要将修正后的内容同步更新到世界书对应条目？"
   - 如果用户同意，使用 Edit 工具更新世界书中的条目
6. 输出提醒：

哥哥，衣柜清单已完成并写入世界书，条目配置已自动设置！
